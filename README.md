# color_map_matlab

=====

## Description

This project introduces many patterns of color map using MATLAB.



## Usage

1. Set Parameters.
2. Excute the "create_color_map" function with the arguments.



## Sample Code

```matlab
titleName = "Google";
colorGradStep = 5;
custom_rgb = {
    [66 133 244]
    [219 68 55]
    [244 180 0]
    [15 157 88]
    };

create_color_map(titleName, colorGradStep, custom_rgb);
```
